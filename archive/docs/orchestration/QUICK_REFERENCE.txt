┌────────────────────────────────────────────────────────────────┐
│ INSTALLATION                                                   │
└────────────────────────────────────────────────────────────────┘
1. Extract zip: unzip langgraph_integration.zip
2. Copy files to project root:
   cp -r langgraph_integration/src/* src/
   cp -r langgraph_integration/config/* config/
   cp -r langgraph_integration/tests/* tests/
   cp langgraph_integration/test_langgraph_parity.py .
3. Run verification: bash INSTALL_VERIFY.sh

┌────────────────────────────────────────────────────────────────┐
│ CONFIGURATION                                                  │
└────────────────────────────────────────────────────────────────┘
Edit config/main.yaml:

workflows:
  use_langgraph: true   # Enable LangGraph mode
  # OR
  use_langgraph: false  # Use sequential mode (default)

┌────────────────────────────────────────────────────────────────┐
│ TESTING                                                        │
└────────────────────────────────────────────────────────────────┘
# Parity test (compares both modes)
python test_langgraph_parity.py

# Unit tests
pytest tests/test_engine_parity.py -v

# Manual test - Sequential
python ucop_cli.py generate --input test.md --output output_seq/

# Manual test - LangGraph (after enabling in config)
python ucop_cli.py generate --input test.md --output output_lg/

# Compare outputs
diff output_seq/index.md output_lg/index.md
