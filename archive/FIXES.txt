USAGE
=====

To build and run the web UI:

1. npm run build      # Auto-removes .npmrc, installs deps, builds React UI
2. python start_web.py # Start the web server
3. Open http://localhost:8000/ in browser

Or for development:

npm run dev           # Auto-removes .npmrc, installs deps, starts Vite dev server

VISUAL WORKFLOW EDITOR
=======================

Access at: http://localhost:8000/

Features:
- Left panel: Drag-and-drop agent palette (34 agents)
- Center canvas: Visual workflow builder with node connections
- Right panel: Job monitor with real-time status
- Top toolbar: Save/Run workflow buttons
